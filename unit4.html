Lesson #9 C.F.U

In your own words describe the benefits of creating functions in your code? Answer- Functions make code more readable, function names allow programmers to explain the goal of sections of code and make it easier to understand a program. Also, functions make code more reusable, functions reduce redundancy by allowing programmers to call functions at different places in a program. Functions allow for abstraction, providing the ability to take specific actions and separate them into reusable parts.
